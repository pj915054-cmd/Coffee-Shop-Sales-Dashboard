This dataset represents transactional sales data from a coffee shop and is designed
for sales performance and customer behavior analysis.

### Columns Description

| Column Name      | Description |
|------------------|------------|
| Order_ID         | Unique identifier for each order |
| Order_Date       | Date when the order was placed |
| Order_Time       | Time of order (Hour level) |
| Customer_ID      | Unique customer identifier |
| Category         | Product category (Coffee, Tea, Chocolate) |
| Product_Name     | Name of the product sold |
| Quantity         | Number of items sold |
| Sales            | Total sales amount for the order |
| Payment_Type     | Mode of payment (Card, Cash, UPI) |
| Time_Period      | Time slot (Morning, Afternoon, Evening, Night) |
| Weekday          | Day type (Weekday / Weekend) |

### Data Usage
- Used for KPI calculations (Sales, Orders, AOV)
- Time-based trend analysis
- Category & product performance analysis
- Payment behavior analysis
- RFM segmentation and Drill Through analysis
